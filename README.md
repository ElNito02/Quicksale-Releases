# QuickSale POS (Releases)

This repository is dedicated to hosting the compiled distribution binaries and auto-updater configuration for the **QuickSale POS** desktop application.

> 🔒 **Security Note:** The source code for QuickSale POS is maintained in a separate, secure private repository. This repository only contains the final, packaged installer files.

## Installation

To install or update the application manually:
1. Navigate to the **[Releases](../../releases)** section of this repository.
2. Download the latest `.exe` installer.
3. Run the installer on your Windows machine.

## Auto-Updates

The QuickSale POS desktop application is configured to automatically check this repository for new versions. When a new version is published here, the application will download it in the background and prompt you to install it when you close the app.
